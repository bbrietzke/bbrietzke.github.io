I always seem to forget which 74XXX-series chips i want to experiment/play with
so i'm going to write them down.

* 74529 :: [8-Bit Addressable Latches](http://www.ti.com/lit/ds/symlink/sn74ls259b.pdf)
* 74245 :: [Octal Bus Transceivers with 3-State Outputs](http://www.ti.com/lit/gpn/sn74abt245b)
*  7467 ::  [Registers 4-by-4 register file With 3-State Outputs](http://www.ti.com/lit/ds/symlink/sn74ls670.pdf)
* 7440105 :: [4-Bit x 16-Word FIFO Register](http://www.ti.com/lit/ds/symlink/cd74hc40105.pdf)
* 74138 :: [ 3-Line To 8-Line Decoders/Demultiplexers](http://www.ti.com/lit/ds/symlink/sn74hc138.pdf)


Others:
